# Tasks

Tasks help you track your work. You can create, assign, and complete tasks
from the dashboard.

## Creating a Task

Go to the **Tasks** tab and click **New Task**. Fill in the details:

- **Title** — short description of the work
- **Assignee** — who is responsible
- **Due date** — when it needs to be done

## Assigning a Task

You can assign a task to any team member. Open the task and click the
assignee field:

![Assign task](/public/Help/images/images.jpeg)

## Status

Tasks have three statuses:

| Status      | Meaning         |
|-------------|-----------------|
| Todo        | Not started yet |
| In Progress | Being worked on |
| Done        | Completed       |

## Using the CLI

You can also create tasks from the command line:

```bash
myapp task create --title "Fix bug" --assignee john
```

For more options see the [CLI reference](/help/cli).
